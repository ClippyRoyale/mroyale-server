# Mario Royale Server [No SQL]
Project to remove most of the MySQL functions from the MR server.

## Python version
Python 3.7 (and up)

## Dependencies
- argon2-cffi (account functions)
- captcha (required if argon2-cffi is installed)
- twisted
- autobahn
- emoji
- jsonschema
- objgraph
- and others if python throws an error
